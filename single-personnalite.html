Introduction

Lea Verou (Grec: Λία Βέρου; née le 13 juin 1986) est une développeur web front-end, conférencière et auteur, originaire de Lesbos, Grèce. Verou est actuellement assistant de recherche au MIT CSAIL, dans le groupe Haystack de David Karger et expert invité dans le groupe de travail CSS du W3C.

Verou a écrit un livre sur les CSS avancés pour O'Reilly, a travaillé pour W3C / MIT, donné plus de 60 conférences invitées à travers le monde, a publié plusieurs projets open source et a co-fondé une startup grecque appelé Fresset Ltd (qu'elle a quitté en 2011), entre autres projets. Elle a écrit des articles pour plusieurs magazines de l'industrie de la conception Web, dont A List Apart et Smashing Magazine.

Education

Verou est titulaire d'un BSc en informatique de l'Université d'Athènes d'économie et de commerce, dans lequel elle a co-organisé un cours de 4e année sur le développement Web dans le passé. Son expérience englobe à la fois le développement technique et la conception visuelle.

Education

Dabblet

Dabblet est une application Web open-source pour le prototypage rapide de HTML, [8] CSS et JavaScript, avec des éclairages de syntaxe et des prévisualiseurs en ligne. Le résultat est enregistré en ligne sur un GitHub Gist pour permettre le partage avec d'autres personnes.

-prefix-free

-prefix-free est un polyfill qui permet aux navigateurs actuels de reconnaître les versions sans préfixe de plusieurs propriétés CSS3 au lieu de demander au développeur d'écrire tous les préfixes du fournisseur. Il lit les feuilles de style de la page et remplace toutes les propriétés sans préfixe par leurs équivalents préfixés reconnus par le navigateur actuel.

Prisme

Prism est une bibliothèque de coloration syntaxique. C'est un projet dérivé de Dabblet. La page du projet a accueilli 23 000 visiteurs uniques le premier jour. Il est utilisé sur des sites Web tels que Smashing Magazine, Mozilla Developer Network et le blog de Brendan Eich.
